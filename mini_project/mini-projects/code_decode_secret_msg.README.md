# Message Decoder (Python)

A Python program that decodes an encoded message using a dictionary-based mapping technique.

## Features
- Uses dictionary for character mapping
- Decodes encoded text back to original message
- Demonstrates reverse dictionary logic

## Concepts Used
- Dictionaries
- Loops
- String replacement
- .maketrans() and .translate()
- Sorting keys

## How to Run
```bash
python message_decoder.py

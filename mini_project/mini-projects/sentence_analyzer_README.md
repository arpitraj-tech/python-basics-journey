# Sentence Analyzer (Python)

A Python program that analyzes a sentence entered by the user and provides useful information about it.

## Features
- Counts total characters
- Counts total words
- Counts vowels and consonants
- Converts sentence to uppercase and lowercase

## Concepts Used
- Strings
- Loops
- String methods
- Dictionaries
- Basic logic building

## How to Run
```bash
python sentence_analyzer.py

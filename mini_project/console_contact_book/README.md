# Python Console Contact Book

A simple console based contact management system built using Python.

## Features
- Add new contacts
- Search contact by name
- Delete contact
- View all contacts
- Input validation for phone numbers
- Modular code using multiple files

## Project Structure
main.py      -> controls menu and flow
contacts.py  -> all contact operations
data.py      -> stores shared contact dictionary

## How to Run
python main.py
